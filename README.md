WayFarer is a public bus transportation booking server

# Features Include

  - User Management system
  - Trip booking management system
  - Roles and privileges
  
# Build
- Node JS
- Express JS
- PostgreSQL (database)
   
# Start App
   ```
   npm install
   create .env using the sample provided in .env.sample
   
   npm run start
   ```
